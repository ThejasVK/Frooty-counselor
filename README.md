## ğŸ¨ Customization

### Colors
The color scheme can be easily modified in `styles.css` by changing the CSS variables:

```css
:root {
    --primary: #6B8E7B;        /* Main green */
    --primary-dark: #4A6B5A;   /* Dark green */
    --secondary: #D4A574;      /* Warm accent */
    /* ... more colors */
}
```

### Content
Update the following in `index.html`:
- Contact information (phone, email, address)
- Social media links
- Testimonials
- Service descriptions
- Working hours

### Images
Replace the placeholder icons with actual images:
1. Add images to an `images/` folder
2. Update the `<img>` tags in `index.html`
